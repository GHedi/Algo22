# Archive Tugas & Praktikum: Algoritma dan Pemrograman (T.A. 2022)

Repositori ini berisi kumpulan dokumentasi tugas, proyek kecil, dan latihan pemrograman yang saya selesaikan selama menempuh mata kuliah Algoritma dan Pemrograman di tahun 2022, program studi Teknik Perangkat Lunak, Universitas Universal.

## 📌 Tujuan Repositori
Dibuat sebagai rekam jejak akademis (*academic archive*) untuk mendokumentasikan pemahaman dasar saya mengenai logika pemrograman, struktur data, dan pemecahan masalah (problem-solving) sejak awal masa perkuliahan.

## 💻 Cakupan Materi & Implementasi Kode
Di dalam repositori ini terdapat beberapa implementasi logika dasar, meliputi:
* **Dasar Pemrograman:** Variabel, tipe data, dan operasi kondisi (If-Else / Switch-Case).
* **Perulangan (Looping):** Implementasi For, While, dan Do-While untuk efisiensi eksekusi logika.
* **Struktur Data Dasar:** Penggunaan Array (1-Dimensi & Multi-Dimensi).
* **Fungsi & Prosedur:** Modularisasi kode untuk penulisan program yang lebih rapi (clean code).

## 🛠️ Tools yang Digunakan
* **Bahasa Pemrograman:** Java / C++ (sesuaikan dengan bahasa yang kamu pakai tahun 2022 dulu)
* **IDE/Teks Editor:** NetBeans / VS Code / Dev-C++
